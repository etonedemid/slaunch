# sLaunch locale template.
#
# English is built in and is always the fallback: any string missing here (or a
# missing file entirely) simply renders in English. To translate the menu:
#   1. Copy this file to  <code>.txt  where <code> is your language code, e.g.
#      fr.txt, de.txt, es.txt, ja.txt, pt-BR.txt (region form is tried first,
#      then the two-letter form).
#   2. Put it in  sdmc:/slaunch/lang/  on the SD card.
#   3. Replace the text on the RIGHT of each '=' with your translation. Leave the
#      LEFT side (the English key) exactly as it is.
# Lines starting with '#' and blank lines are ignored. The console's system
# language selects which file is loaded.
