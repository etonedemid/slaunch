# Bundled homebrew loader

`main` / `main.npdm` here are a build of **[nx-hbloader](https://github.com/switchbrew/nx-hbloader)**
(© 2017-2018 nx-hbloader Authors, ISC license). sSystem serves this folder as
external code for the album applet slot via ECS when you open the Homebrew menu;
launched with no target, nx-hbloader loads `sdmc:/hbmenu.nro`.
